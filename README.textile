TextMate Bundle for writing YARD style documentation for ruby scripts

h4. Snippets:

  * @par - @params
  * @si  - @since
  * @ver - @version
  * @au  - @author
  * @see - @see
  * @rai - @rai
  * @ypar - @yieldparam
  * @y   - @yield
  * @ret - @return
  * @dep - @deprecated
